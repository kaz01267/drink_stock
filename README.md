# DrinkStock

## サービス概要

DrinkStockは、家にあるお酒の在庫を管理し、飲酒履歴を記録できる個人向けWebアプリです。
記録されたデータをもとに、「最近飲んでいない」「在庫が多い」といった観点から、
今日は何を飲むかを考える時間を楽しくする提案を行い、将来的には感想や思い出も残せるようにします。

## このサービスへの思い・作りたい理由

家にお酒の在庫が増えてくると、「何がどれくらい残っているのか分からない」「結局いつも同じお酒を飲んでしまう」といった状況になりがちでした。
また、過去に飲んだお酒の感想や、その時の出来事を振り返りたいと思っても、記録として残っていないことに不便さを感じていました。

そこで、在庫管理と飲酒記録を一元化し、
「今日は何を飲もうか」と考える時間そのものを楽しめるサービスを作りたいと考え、本サービスを企画しました。

## ユーザー層について

- 自宅でお酒を飲むことが多い人

- 複数種類のお酒をストックしている人

- お酒の記録や思い出を残したい人

- 在庫管理と飲酒記録をシンプルに行いたい個人ユーザーを主な対象としています。

## サービスの利用イメージ

ユーザーは、家にあるお酒を在庫として登録し、飲んだ際に飲酒記録を残します。
飲酒記録に応じて在庫数量が自動的に更新されるため、現在の在庫状況を把握できます。
将来的には、記録されたデータをもとに「今日は何を飲むか」を提案する機能や、感想・思い出を振り返る体験を提供します。

## サービスの差別化ポイント・推しポイント

- 在庫管理と飲酒記録を一体化している点

- 個人利用に特化したシンプルなUI

- 将来的に「思い出を残す」「振り返る」体験を重視している点

- 単なる在庫管理に留まらず、お酒にまつわる体験そのものを記録・可視化する点を推しポイントとしています。

## 機能候補

### MVPリリース時に作りたい機能（最小構成）

- ユーザー登録 / ログイン

- お酒の在庫登録・編集・削除

- 飲酒記録の登録

- 飲酒記録に基づく在庫数量の更新

### 本リリース・将来的に追加したい機能

- シンプルな「今日のお酒」提案表示

- 提案条件の切り替え（最近飲んでいない / 在庫が多い / ランダム）

- 飲んだお酒への感想・思い出の記録

- 飲酒履歴の一覧・振り返り機能
